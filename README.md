# Calculator App 🧮

A fully functional calculator built with pure HTML, CSS, and JavaScript.

## 🔗 Live Demo
https://manohar-web-developer.github.io/calculator-project_by_manohar/
## ✨ Features
- Basic operations: Addition, Subtraction, Multiplication, Division
- Percentage calculation
- Decimal point support
- Backspace & Clear functionality
- Expression display (shows equation on top)
- Fully responsive (320px to 1440px)

## 🛠️ Built With
- HTML5
- CSS3 (Grid, Box Shadow, Media Queries)
- Vanilla JavaScript (DOM Manipulation)

## 📚 What I Learned
- DOM manipulation using `getElementById` and `querySelector`
- Event listeners and callback functions
- JavaScript logic building (if/else conditions)
- String methods (`slice`, `includes`)
- `parseFloat` for string to number conversion
- CSS Grid for button layout
- 3D button effect using `box-shadow` and `linear-gradient`
- Responsive design using Media Queries
